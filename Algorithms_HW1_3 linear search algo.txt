
Задание:
Написать псевдокод для алгоритма линейного поиска
Реализовать алгоритм в коде


Решение (псевдокод):
1. Создать функцию linearSearch с параметрами array (массив) и target (целевое значение).
2. Итерироваться по индексам i от 0 до длины массива минус 1.
3. Проверить, равен ли элемент array[i] целевому значению target.
4. Eсли элемент равен целевому значению, вернуть индекс i.
5. Eсли все элементы были просмотрены и целевое значение не было найдено, вернуть -1.
