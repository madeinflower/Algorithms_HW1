
Задание:
Написать псевдокод для алгоритма: сложить три числа и вывести сумму
Реализовать алгоритм в коде Java

Решение (псевдокод):
START
    // Ввод трех чисел
    READ num1
    READ num2
    READ num3
    
    // Вычисление суммы
    sum = num1 + num2 + num3
    
    // Вывод суммы
    PRINT sum
END

Код в Java отправлен отдельным файлом
